Usable Design Analysis

Posted by Jon Hetrick on April 19, 2020

Post #2

Analysis of link www.mrbottles.com

We will be taking a look this random website and seeing if we can analysis to to some certain criteria and possibly spot some ways to make it better for the end user that is visiting the website.



What we will be looking at:

  • What is the purpose of this website?
  • Are the Navigation elements clear?
  • Are there clear navigation hierarchies?
  • Is there more content to be found if I scroll?


What is the purpose of this website?

My initial thought of the page is, "yes, we are on a website about bottles." So, I would say that it is the first win and possibly the last one. Because beyond that, we have no idea if we are buying bottles or is this a museum of different bottles? Eventually we are able to see the line "Antique Bottle Collectors Resource," honstly I was moving on to the next creteria before I even saw that.



Are the Navigation elements clear?

Sort of... Let's dig in:

  1. The is no title in the top left, it is in the middle
  2. The naming of the Primary Navigation is kind of wierd. The first one we see is Gallaries and Of Interest.
  3. The secondary Navigation is on the bottles? Is there a link there on the "Milk" Bottle, I can barely make that out.
  4. When Scrolling down the webpage, there is "Back to the Surface Button" that follows you, but the icon covers the text. If I were reading, I would have to scroll to get it out of the way.
  5. The footer nav has the same links as the thought to be primary nav?
  6. "The Need An Appraisal?" button must go!


Are there clear navigation hierarchies?

Overall the hierarchies are clear, the primary looks to be in the correct spot. The secondary nav is on the bottles. The less essential links are in the footer nav. There is a Logo/Home on the page. But, let us be honest; it hurts visually navigate.



Is there more content to be found if I scroll?

There is a massive amount of content! It is hard to read inter mix with a lot of images. There are links and it is basically a on going living blog post / home page. We can make this better!

Image of a slice from the Home Page:



Ideas to fix this up:

The website just needs that clear once over. Taking all of the basics that we have learned thus far would make this website perfect fine. I would Take all the navigation out to start and:

  1. Add the logo and heading into the top left.
  2. Rename the confusing primary headings.
  3. Add all the Secondary Links that are on the bottles to new Primary Nav Link in a dropdown.
  4. Change the Home Page Image to something with no animations.
  5. Move the Need An Appraisal? Call to Action else where on the Page, maybe that could be in the center. Again without the Pig.
  6. If there is anything left that seems not very important, sure throw that into the footer nav.
  7. Remove everything else from the home page and put that into new pages. The about us, blog post, etc.
  8. Change the color palet. Keep the Antique Vibe, but maybe more modern with some whites with the brown instead of the black.
  9. Change the typography
